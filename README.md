# 🚀 Flask Login & Dashboard System

A simple and modern Login Authentication System built using **Flask**, **HTML**, **CSS**, and **JavaScript**.
This project includes:

* User Signup 🔐
* User Login 👨‍💻
* JWT Authentication 🛡️
* Session Management 🍪
* Dashboard UI 📊
* Logout Functionality 🚪
* Forgot Password Demo ✉️

# 🔑 Default Workflow

## Signup

1. Create account
2. Password gets hashed
3. User stored in mock database

## Login

1. Validate email & password
2. Generate JWT token
3. Store token in session

## Dashboard

1. Protected using authentication
2. Dashboard data loaded from API

## Logout

1. Session token removed
2. Redirect to login page
